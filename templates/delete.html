/delete Endpoint

This endpoint is used to delete an education/employment/certification from a user's blockchain.

DELETE Request

To delete, this endpoint is expecting a DELETE request with a JSON body.

JSON Format

{
    "userId": [string],
    "certificate_token": [string],
    "certificate_url": [string],
    "certificate_category": [string],
    "institution_name": [string],,
    "degree_name": [string],,
    "date_range": [string],,
    "description": [string],,
    "index": [int],
    "timestamp": [int],
    "previousHash": [string],,
    "hash": [string],,
    "blockType": [string],,
    "createOriginHash": [string],
}
