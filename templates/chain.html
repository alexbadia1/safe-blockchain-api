/chain Endpoint

This endpoint is used to retrieve a user's entire blockchain.

GET Request

To retrieve a user's blockchain, this endpoint is expecting a GET request with the following query parameters:

  • {userId} = [int] -> users blockchain id number.
  • {view} = raw | filtered [default] -> whether to show genesis, update, and delete blocks.

Format

  • https://safe-demo-api.herokuapp.com/chain?userId={int}&view={raw | filtered [default]}

Examples

  • Filtered Blockchain for userId 1: https://safe-demo-api.herokuapp.com/chain?userId=1

  • Filtered Blockchain for userId 2: https://safe-demo-api.herokuapp.com/chain?userId=2&view=filtered

  • Raw Blockchain for userId 3: https://safe-demo-api.herokuapp.com/chain?userId=3&view=raw
