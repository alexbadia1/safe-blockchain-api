Chain Endpoint

This endpoint is used to retrieve a user's entire blockchain.

GET Request

To retrieve a user's blockchain, this endpoint is expecting a GET request with a JSON body.

JSON Format

{
    "userId" : <string>
}

Example

GET Request

{
    "userId" : "1",
}
