/mine Endpoint

This endpoint is used to mine a block on the user's blockchain.

POST Request

To mine, this endpoint is expecting a POST request with a JSON body.

JSON Format

{
    "userId": [string],
    "certificate_token": [string],
    "certificate_url": [string],
    "certificate_category": [string],
    "institution_name": [string],
    "degree_name": [string],
    "date_range": [string],
    "description": [string],
    "index": [int],
    "timestamp": [int],
    "previousHash": [string],
    "hash": [string],
    "blockType": [string],
    "createOriginHash": [string],
    "nonce": [int]
}
